# Marketing Playbook

## Positioning statement

For **regulated enterprise AI platform teams** who cannot safely scale agents to production, **AegisAI** is the **agent governance control plane** that discovers, governs, approves, executes, and audits agentic work across any framework — unlike agent builders or observability-only tools.

## Content pillars

1. **Agent sprawl is the new shadow IT** — registry, ownership, cost  
2. **Propose ≠ execute** — architecture and gateway pattern  
3. **Audit-ready AI** — audit packets, EU AI Act mapping  
4. **FinOps for agents** — cost anomalies, ROI per workflow  
5. **Safe velocity** — eval gates + HITL without killing automation

## Campaign ideas

### “Gateway Week”

Release Python + TS SDK; daily code samples for OpenAI Agents, LangGraph, MCP proxy pattern.

### “Audit packet challenge”

Post a anonymized refund case PDF; ask “Can your stack produce this in one click?”

### “Agent inventory scorecard”

Free spreadsheet template → upsell to registry product.
